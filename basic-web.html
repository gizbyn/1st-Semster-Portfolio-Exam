Behind About

... basic web -> coding, designing, responsiveness.

General Information


Theme number: 01
Semester: 1st semester
Duration: 4 weeks
ECTS: 6 ECTS

Purpose


In this theme, I was introduced to the basic terminology within the user interface design, digital communication, user testing, and responsive web design. I also gained skills in HTML and CSS coding as well as basic image processing (Photoshop) and vector drawing (Illustrator).

Project

01.04.01 Responsive website
Work process
This project was the biggest project of the 1st theme of the 1st semester - Basic Web. Before actual coding, I was assigned a specific style - Punk. I must admit that it is not one of my favourite styles, but at least I have got to know something new. I needed to create a moodboard, then a design and process document with my style tile (colors, fonts, ...), different wireframes for different devices, my logo, used gestalt laws, XD Prototype.

My reflections
Personally, I consider this theme and everything I have learnt as very useful. However, I struggled a lot, especially with working in Adobe Products, because a lot of new features were introduced to me. I also appreciate the insight into coding. Despite the fact that it is very difficult and even a small mistake (like forgetting to type a semi-colon at the end of the command) can ruin your code, I find it very useful.
Even though it was the first theme, I am satisfied with the results and I believe that they have value not only for me.

Skills gained
The skills I gained during this project and theme include knowledge of colour theory, typography, design principles, HTML and CSS coding, prototyping, working with Adobe apps (Photoshop, Illustrator, XD, InDesign), prototyping and user tests.
